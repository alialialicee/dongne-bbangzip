# 칠일금기

사용자의 이름, 생년월일, 태어난 시간, 성별을 바탕으로 앞으로 일주일 동안 “피해야 할 것”만 기록해 보여주는 모바일 우선 오컬트 공포 콘셉트 웹앱입니다. 결과는 일반적인 운세가 아니라 사용자의 이름으로 열린 금기 문서처럼 구성됩니다.

## 기술 스택

- React
- TypeScript
- Vite
- CSS

## 설치 방법

```bash
npm install
```

## 실행 방법

```bash
npm run dev
```

Vite 개발 서버가 안내하는 로컬 주소로 접속하세요. `index.html`을 직접 여는 방식으로 실행하지 않습니다.

## 빌드 방법

```bash
npm run build
```

## 동작 방식

- 외부 API를 사용하지 않습니다.
- 외부 이미지, CDN, 외부 폰트를 사용하지 않습니다.
- 모든 시각 요소는 CSS와 HTML 요소로 표현합니다.
- 입력값은 서버로 전송하지 않으며, 브라우저 화면에서 결과를 생성하는 용도로만 사용합니다.
- 요일별 금기는 seed 기반 로직으로 생성되며 시간대, 장소, 사람, 물건, 행동, 문장 템플릿을 독립적으로 선택해 7일 내내 같은 조건이 반복되지 않도록 보정합니다.

## Android 앱 변환 메모

최종 목표는 이 웹앱을 추후 Capacitor로 감싸 Android 앱으로 변환하고 Google Play Store 등록에 맞게 확장하는 것입니다. 현재 구조는 모바일 우선 반응형 UI와 정적 프론트엔드 동작을 전제로 구성되어 Capacitor 래핑에 적합합니다.
